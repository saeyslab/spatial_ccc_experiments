# Allen brain atlas 

Zhang et al, 2023

## Requirements

```
pip install spatialdata
pip install scanpy
pip install pooch
```

# Downloading data

`python3 download_allen_brain_atlas.py /path/to/save/`

# Preprocessing data

`python3 preprocess_allen_brain_atlas.py /path/to/save/ [SECTION_NR]`

- `SECTION_NR`: For subsetting of slice section (Optional)
